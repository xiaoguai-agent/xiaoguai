## What the user still owns

These items can't be done in a worktree without a live browser /
terminal recording:

- Capture the 8 screenshots into `docs/screenshots/` per the
  PLACEHOLDER manifest (spin up `pnpm -F chat-ui dev` +
  `pnpm -F admin-ui dev` first).
- Record the 3 asciinema casts into `docs/asciinema/` per the
  PLACEHOLDER manifest, then run `asciinema upload` if a public link
  is wanted.
- Once captured, swap the inline `<!-- screenshot: ... -->` HTML
  comments in `README.md` for real `![alt](docs/screenshots/...)`
  references. Same for asciinema links.
- Eyeball-pass the README on GitHub (the ASCII diagram renders
  differently outside a fixed-width terminal — confirm it survives
  the GitHub markdown renderer).
