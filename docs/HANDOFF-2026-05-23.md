- Every milestone closes with a doc update + `cargo test --workspace`

## 7. Known sharp edges

- `xiaoguai-core serve` falls back to **MockBackend** when the
  `llm_providers` table is empty. Use `xiaoguai provider register` to
  populate it before running real traffic.
- `auth: None` and `authz: None` are the defaults. Set
  `XIAOGUAI_AUTH__REQUIRED=true` + provide JWKS / issuer / audience to
  flip both JWT verification and Casbin per-route enforcement on.
- The LlmBackend impl on LlmRouter uses `ResolveCtx::default()`, so the
  agent loop only sees the system-default + fallback chain — per-tenant
  routing requires a separate slice (deferred to v0.6.3).
- Feishu integration's signature algorithm matches the *common* spec
  variant; some Feishu apps use a slightly different scheme depending on
  encryption settings. Real-world testing pending.
- The 51 ignored tests are mostly testcontainer-based PG e2e — they pass
  locally if you have Docker but are not part of CI's fast path.
